types of variable in class base programming

there 3 types of variables

  • Local variable

    • The Variable which is created inside the method is known as local variable.

  • Static variable

    • The Variable which is created inside the class and outside the method with Static keyword is known as Static variable.

  • Non-Static Variable

    • The Variable which is created the class and outside the method without using Static keyword is known as Non-static variable.
